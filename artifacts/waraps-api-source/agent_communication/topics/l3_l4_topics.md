# L3-L4 Topics
For more information on what topics are see [L1 Topics](l1_topics.md)

The topics that are necessary for an L3-L4 agent to work with most Core System Services are:<br>
• [heartbeat](agent_communication/topics/heartbeat.md) - All agents<br>
• [sensor_info](agent_communication/topics/sensor_info.md) - Level 1 and above agents<br>
• [sensor](agent_communication/topics/sensor.md) - Level 1 and above agents<br>
• [direct_execution_info](agent_communication/topics/direct_execution_info.md) - Level 2 and above agents<br>
• [tst_execution_info](agent_communication/topics/tst_execution_info.md) - Level 3 and above agents<br>
