# SysCraft Arena Runtime - Initial Coding Roadmap

## Milestone 1

Build a Docker-based development environment containing:

-   React + Cesium frontend
-   FastAPI gateway
-   Python simulation kernel
-   MQTT broker
-   PostgreSQL/PostGIS

## Milestone 2

Implement:

-   entity registry
-   simulation clock
-   fixed-step scheduler
-   MQTT interface
-   plugin loader

## Milestone 3

Render moving entities on Cesium.

## Milestone 4

Generate proximity, interaction and collision events.

## Milestone 5

Support dynamically loaded plugins contributing:

-   entity types
-   simulation systems
-   UI panels
-   visual assets
-   configuration

## Milestone 6

Prepare architecture for future Rust ECS migration and HLA federation.
